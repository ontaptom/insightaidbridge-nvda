InsightAid Bridge v0.1.0

NVDA Bridge Addon for InsightAid AI Accessibility Assistant

What is InsightAid Bridge?

InsightAid Bridge is an NVDA addon that captures webpage content and element data, on user request, for use with the InsightAid AI accessibility assistant. This bridge addon provides the NVDA integration layer, while the separate InsightAid application handles AI processing for enhanced accessibility features.

Note: This addon captures and saves data to files only on specific user request - based on predefined shortcuts. The separate InsightAid application processes this data using local AI models to provide features like image analysis, text translation, and document summarization.

Features

  • Element Capture: Capture details about the current focused element including content, role, and location
  • Document Analysis: Extract structured content from entire webpages with proper formatting
  • Privacy-First: All data is stored locally on your computer
  • AI-Ready Format: Outputs structured JSON data optimized for AI processing

Keyboard Shortcuts

  • NVDA+I - Capture current element (image, text, button, etc.)
  • NVDA+Shift+I - Capture full document structure

How to Use

  1. Navigate to any webpage using NVDA
  2. To analyze a specific element (image, heading, etc.), navigate to it and press NVDA+Alt+I
  3. To get a summary of the entire page, press NVDA+Alt+D
  4. You'll hear confirmation beeps when data is captured
  5. The InsightAid application will process this captured data

File Storage

Captured data is automatically saved to:

  • Element data: %appdata%\InsightAid\current_element.json
  • Document data: %appdata%\InsightAid\document_content.json

Requirements

  • NVDA 2021.1 or later
  • Windows 10 or later
  • InsightAid application (for AI processing)
Important: This addon only captures and saves data. To get AI-powered descriptions, translations, and analysis, you need to install the separate InsightAid application.

Troubleshooting

Shortcuts not working?

  • Restart NVDA after installing the addon
  • Check if shortcuts conflict with other addons in NVDA Menu → Preferences → Input Gestures
  • Ensure you're in browse mode when using the shortcuts

No confirmation beeps?

  • Check NVDA's audio settings
  • Verify the addon is enabled in NVDA Menu → Tools → Add-ons Manager

Privacy

InsightAid Bridge respects your privacy:

  • All data is stored locally on your computer
  • No information is sent to external servers
  • You have full control over the captured data
